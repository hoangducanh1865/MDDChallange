Thông tin các file dữ liệu:
1. train.csv: File csv với các cột dữ liệu: ```id,path,canonical,transcript```. 
Trong đó cột canonical và transcript ở dạng readable sentence. 
2. train_phones.csv: File csv tương tự như trên. Trong đó cột canonical và transcript ở dạng phonemes, dấu $ phân biệt giữa mỗi từ.
3. lexicon_vmd.txt: Từ điển phiên âm của các từ nằm trong corpus dữ liệu VMD-VLSP23
